# LX Landing — WordPress Child Theme

LX Landing là một child theme cao cấp dựa trên **Hello Elementor**, được tối ưu hóa để xây dựng các trang Landing Page nhanh chóng, hiệu quả và bảo mật. Theme tích hợp sẵn hệ thống Theme Options mạnh mẽ và các Widget Elementor tùy chỉnh.

## Tính năng nổi bật
- **Theme Options (ACF Pro):** Quản lý font chữ (Google Fonts), màu sắc chủ đạo và cấu hình nút bấm toàn trang.
- **Custom Widgets:** Hệ thống hơn 20+ widgets Elementor độc quyền (Tiêu đề, Banner, Dịch vụ, Đánh giá, FAQs...).
- **Security Hardening:** Tích hợp bộ công cụ bảo mật (ẩn phiên bản WP, tắt Emoji, dọn dẹp header, khóa file mod...).
- **Custom Code Manager:** Dễ dàng chèn mã theo dõi (GTM, Pixel, Analytics) vào Header, Body hoặc Footer.
- **Auto Update:** Hỗ trợ kiểm tra và cập nhật theme tự động từ GitHub.

## Yêu cầu hệ thống
- WordPress 6.0 trở lên.
- PHP 8.0 trở lên.
- **Plugins bắt buộc:** Elementor, ACF Pro 6.2+, Pro Elements (hoặc Elementor Pro).

## Cấu trúc thư mục
- `/inc`: Chứa các module xử lý logic (hàng đợi assets, đăng ký CPT, bảo mật...).
- `/widgets`: Chứa mã nguồn các widget Elementor tùy chỉnh.
- `/assets`: Chứa các file SCSS, JS và thư viện đi kèm.
- `/templates`: Chứa các file template part tái sử dụng.

## Cài đặt
1. Tải bộ code lên thư mục `/wp-content/themes/`.
2. Kích hoạt theme trong giao diện Admin.
3. Cài đặt các plugin được yêu cầu tại mục **Cài đặt > Plugin cần thiết**.
